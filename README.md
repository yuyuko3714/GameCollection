# GameCollection

本课题设计并实现了一个基于Java 21 LTS和Swing GUI框架的小游戏合集平台，包含三款风格各异的独立游戏：回合制策略游戏《植物王国崛起》、休闲消除游戏《炸炸乐》以及动作射击游戏《弹幕贪吃蛇》。系统采用模块化设计，通过统一的主菜单界面实现游戏间的无缝切换，各游戏独立封装且具备完整的游戏逻辑、UI界面、音效系统和事件机制。

《植物王国崛起》采用MVC架构，玩家通过选择不同天赋策略，管理植物种群的能量、繁殖与生存，在随机事件系统中挑战46回合的生存目标；《炸炸乐》融合扫雷棋盘规则与连锁爆炸消除机制，支持娱乐模式与策略模式双玩法；《弹幕贪吃蛇》实现了高帧率的实时游戏循环，结合贪吃蛇尾巴机制与弹幕躲避玩法。系统整体代码量约3000行，采用面向对象设计思想，充分运用Java的多线程、事件驱动、泛型、枚举等特性，展现了Java语言在桌面应用开发中的强大能力。



关键词： Java Swing；游戏开发；MVC架构；面向对象；事件驱动
